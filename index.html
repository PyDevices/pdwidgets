PyDevices / pdwidgets

Cross-platform widgets for pydisplay

pdwidgets is a lightweight widget toolkit for pydisplay: buttons, lists, forms, gauges, themes, and navigation patterns that run on MicroPython and CPython without adopting a full LVGL stack. It targets demos and apps that need interactive chrome on top of pydisplay's display and event layers.

Install from the PyDevices micropython-lib MIP index or TestPyPI, then import pdwidgets alongside board_config. The PyScript gallery pulls it at runtime for widget demos — the same packages you would MIP onto a board — so what you try in the browser stays honest to the device path.

import mip
