PyDevices / pdwidgets

Widgets for pydisplay — without adopting LVGL.

pdwidgets is a lightweight widget toolkit for pydisplay: buttons, forms, dialogs, navigation, sheets, gauges, and charts that run on MicroPython and CPython. It sits on Display + Screen and drives the loop through board_config.runtime.run_forever().

Examples live in the pydisplay tree as src/examples/widgets_*.py and show up in the PyScript gallery via MIP — the same packages you install on a board.

PyScript gallery Documentation GitHub

Install

MIP index + lean imports for MCU 
import mip
mip.install("pdwidgets", index="https://PyDevices.github.io/micropython-lib/mip/PyDevices")

# Prefer lazy imports on constrained boards:
from pdwidgets.widgets.button import Button

Hello Button

Minimal Display / Screen / Button loop.

import board_config
import pdwidgets as pd

display = pd.Display(board_config.display_drv, board_config.runtime)
screen = pd.Screen(display)
pd.Button(screen, x=10, y=10, w=120, h=40, label="Hello")
board_config.runtime.run_forever()

Widget categories

not every class — see docs for the full surface

Forms & controls

Button, Switch, CheckBox, Slider, TextInput, Dropdown, NumberStepper, RadioGroup, Form / FormRow.

Navigation

AppBar, Page, Navigator, TabView / TabBar, Menu, ListView, SegmentedControl.

Sheets & overlays

Dialog, Drawer, BottomSheet, Toast, Accordion, Card, Keyboard.

Charts & feedback

Chart, Gauge, ProgressBar, Spinner, Badge, DigitalClock, and layout helpers (Row, Column, Grid).

Interactive examples ship with pydisplay as src/examples/widgets_demo.py, widgets_settings.py, widgets_nav_tabs.py, and more — not inside this repo. Themes pull Material colors through palettes (ColorTheme + material_design).
